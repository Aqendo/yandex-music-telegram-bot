# Yandex Music Telegram Bot

Это бот, который выдаёт MP3 файл музыки, которую сейчас слушает пользователь в Яндекс Музыке

## Установка
### Установите зависимости

```bash
$ pip install -r requirements.txt
```

### Измените .env
```bash
$ cp .env.example .env
$ nano .env
```

### Запустите бота
```bash
$ python3 main.py
```
